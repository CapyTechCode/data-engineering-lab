# 🦫 CapyTech — Vault de conocimiento

Bienvenido al vault de Obsidian del curso de Data Engineering.

## Cómo usar este vault

1. Descarga [Obsidian](https://obsidian.md) (gratis)
2. Click en "Open folder as vault"
3. Selecciona esta carpeta (`obsidian-vault/`)
4. Activa los plugins recomendados cuando te lo pida

## Estructura

- `00-mapa/` — índice del curso y roadmap
- `videos/` — una nota por cada episodio del canal
- `conceptos/` — notas atómicas de cada concepto (Zettelkasten)
- `sintaxis/` — cheatsheets de referencia rápida
- `proyectos/` — guías de proyectos prácticos
- `templates/` — plantillas para crear tus propias notas

## Cómo navegar

Usa `Ctrl+Click` en cualquier `[[link]]` para saltar a esa nota. Abre el Graph View (`Ctrl+G`) para ver todas las conexiones. Cada concepto te dice en qué videos aparece y cómo se relaciona con otros.

## Actualizar

```bash
cd data-engineering-lab
git pull
```

Las notas nuevas aparecerán automáticamente en tu vault.
